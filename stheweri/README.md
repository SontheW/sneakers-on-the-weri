# stheweri — StheW (Sneakers on the Weri)

Projeto Next.js + TypeScript (starter) — Dark-first with manual theme toggle.

## Como testar localmente (PC/Mac)
1. Instale Node.js (LTS) e Git.
2. No terminal: `git clone <YOUR-REPO-URL>`
3. `cd stheweri`
4. `npm install`
5. `npm run dev`
6. Abra http://localhost:3000

## Como subir para o GitHub (comandos)
```bash
git init
git add .
git commit -m "Initial commit - stheweri"
git branch -M main
git remote add origin https://github.com/YOUR-USER/stheweri.git
git push -u origin main
```

## Deploy no Vercel
1. Crie conta em https://vercel.com e conecte seu GitHub.
2. New Project → importe o repositório `stheweri` → Deploy.
3. Vercel publica automaticamente.
